## Commands
`(optional) [required]`


- `!roll [count]d[die_size](modifier)`
- `!lookup [category] [name]`

## Examples

### Roll

```
!roll 1d6+2
```

### Lookup

```
!lookup weapon lonsword


!lookup w longsword
```


```
!lookup monster Aboleth
```
